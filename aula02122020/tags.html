Elementos HTML

O que é o HTML?

O HTLM (Linguagem de Marcação de HiperTexto) é uma linguagem de hipertexto e marcação, e esta é sua principal característica, o que a torna tão poderosa para desenvolvimento de sites. As marcações são inseridas nos textos, porem para os olhos dos leitores elas são invisíveis, pois sua principal finalidade é de passar instruções ao navegador para que o mesmo possa renderizar uma pagina.

O que são TAGs html?

As TAGs são estruturas de linguagem de marcação que contém instruções para renderização no navegador. Os navegadores interpretam essas instruções e a partir disso renderizam uma pagina. Podemos ainda dizer que "As tags também são responsáveis por criar elementos nas paginas web com a função de organizar, identificar e limitar o conteúdo de uma pagina. Cabe então ao Browser reconhecer esses elementos, e renderizá-las apresentando seu conteúdo aos usuários."

Para saber mais sobre TAGs e elementos do HTML visite Elementos HTML e conheça todos os elementos aplicáveis.
